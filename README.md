    - 2021-09-13
    - 매개변수
      - src
        - 복사할 배열 이름
      - srcPos
        - 복사할 배열의 첫 번째 위치
      - dest
        - 복사해서 붙여 넣을 대상 배열 이름
      - destPos
        - 복사해서 대상 배열에 붙여 넣기를 시작할 첫 번째 위치
      - length
        - src에서 dest로 자료를 복사할 수 있습니다.
